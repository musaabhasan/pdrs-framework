# Roadmap

## Phase 1: Foundation

- Event landing pages.
- Email OTP and signed-link verification.
- Encrypted registrations.
- Moodle user creation and enrollment.
- Course and cohort mapping.
- SMTP notifications.
- Calendar invite generation.
- Audit logging.
- Rate limiting.

## Phase 2: Administration

- Admin authentication with MFA.
- Event builder UI.
- Custom field designer.
- Approval queue.
- Moodle provisioning retry queue.
- Audit log viewer.
- Export controls.

## Phase 3: Payments and Policy Automation

- Payment provider integration.
- Domain policy management.
- Registration capacity rules.
- Waiting list support.
- Conditional enrollment by registration type.

## Phase 4: Resilience and Scale

- Queue worker for Moodle API operations.
- Reminder scheduler.
- Horizontal application scaling.
- Centralized observability.
- Backup and restore automation.
- Data retention automation.

## Phase 5: Analytics

- Conversion funnel.
- Registration source tracking.
- Enrollment success dashboard.
- Program attendance integration.
- Executive reporting exports.
